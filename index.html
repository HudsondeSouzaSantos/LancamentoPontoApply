Lançamento de Ponto

Este site foi desenvolvido para preenchimento automático das folhas de ponto disponibilizado pela Apply.

Para correta utilização, deve-se primeiramete enviar os arquivos de imagem para o Hudson e também o arquivo JS com seus dados para disponibilizar no combobox abaixo.

Após esse procedimento seu nome irá aparecer no combo abaixo e basta selecionar para imprimir sua folha de ponto.
